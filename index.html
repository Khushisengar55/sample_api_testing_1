Problem Statement

Find the highest salary that was credited to an employee, but only for transactions that were not made on the 1st day of any month. Extract:

  • SALARY: Highest credited salary (excluding day 1)
  • NAME: Combined First & Last name (e.g. Emily Brown)
  • AGE: Precise age dynamically computed from DOB
  • DEPARTMENT_NAME: Employee's department

Simulated Schema:

DEPARTMENT EMPLOYEE PAYMENTS
Production-Grade ANSI SQL Query 
SELECT 
    p.AMOUNT AS SALARY,
    CONCAT(e.FIRST_NAME, ' ', e.LAST_NAME) AS NAME,
    (EXTRACT(YEAR FROM CURRENT_DATE) - EXTRACT(YEAR FROM e.DOB) - 
     CASE 
         WHEN EXTRACT(MONTH FROM CURRENT_DATE) < EXTRACT(MONTH FROM e.DOB) 
              OR (EXTRACT(MONTH FROM CURRENT_DATE) = EXTRACT(MONTH FROM e.DOB) 
                  AND EXTRACT(DAY FROM CURRENT_DATE) < EXTRACT(DAY FROM e.DOB)) 
         THEN 1 
         ELSE 0 
     END) AS AGE,
    d.DEPARTMENT_NAME
FROM PAYMENTS p
JOIN EMPLOYEE e ON p.EMP_ID = e.EMP_ID
JOIN DEPARTMENT d ON e.DEPARTMENT = d.DEPARTMENT_ID
WHERE EXTRACT(DAY FROM p.PAYMENT_TIME) != 1
ORDER BY p.AMOUNT DESC
LIMIT 1;
